  • HTML element는start tag부터 end tag까지의 모든 것이다.


HTML Element Syntax
  • HTML element는start tag/opening tag로 시작된다.
  • HTML elements는 end tag/closing tag로 끝난다.
  • Element content는 두 tags 사이의 모든 것이다.
  • 어떤 elements는 content가 없을 수도 있다.
  • Empty elements는 start tag에 닫힌다.
  • 대부분의 HTML elements는 attributes를 가질 수 있다.


Nested HTML Elements
  • 대부분의 HTML Elements는 nested될 수 있다. HTML 문서란 것은 nested HTML elements로 구성되어 있다.

Don’t’ Forget the End Tag

  • 몇몇 HTML elements들은 end tag를 빼먹어도 정확히 보일지도 모른다.
  • <p>this is a paragraph
  • <p>this is a paragraph
  • End tag가optional하게 간주되기 때문에 위 코드는 브라우저 상에서 제대로 나온다.
  • 그러나 절대로 이런 경우에 의존하면 안된다. 왜냐하면 end tag를 붙이지 않았을 경우에 대다수의 HTML elements들은 예상치 못한 error를 발생시킨다.

Empty HTML Elements

  • Content가 없는HTML elements는 empty elements라고 불린다.
  • <br> 이closing tag없는 empty elements 중에 하나이다. (그러나 XHTML, XML의 경우를 고려해서라도 <br> tag는 <br />로 쓰는 편이 낫다.)

HTML Tip: Use Lowercase Tags

  • HTML tags는 대소문자를 구분하지는 않지만 웹 표준이기 때문에 lowercase를 쓰는 것을 권장한다.

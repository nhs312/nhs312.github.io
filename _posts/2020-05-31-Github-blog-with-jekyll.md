## jekyll
테마가 다양하고 github을 지원하는 기본 테마도 있는 등 서포트가 쉬울것이라 판단함
대략의 순서는 아래와 같음
1. mac os에 ruby 설치
2. jekyll 설치
3. 원하는 theme download
4. 로컬에서 bundler를 통해 블로그 셋팅
5. github에 푸시

github surpported theme이 아니라면 gem 베이스가 아니라 일반 thme으로 전환해야 한다. [Themes | Jekyll • Simple, blog-aware, static sites](https://jekyllrb.com/docs/themes/#converting-gem-based-themes-to-regular-themes)
github에 나오는 페이지에서는 이 bundler를 통해 블로그를 보여주지 않기 때문이라서 로컬에 fork를 하든지 해서 셋팅을 다 끝내놓고 push하는 형태로 해야만 했던것.

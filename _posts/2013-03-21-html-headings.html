HTML Headings
  • Heading은 <h1>~<h6>으로 정의된다.
  • <h1>은 가장 중요한 heading을 정의한다. <h6>은 가장 덜 중요한 heading을 가리킨다.


Heading are Important
  • HTML Heading은 오직 heading을 위해서만 쓰인다. 이 heading을 다른 용도로 쓰면 안된다. 검색 엔진은 이러한 heading을 통해서 웹 페이지를 indexing 한다. 때문에 header tag가 많아도 <h1>이 가장 중요하고 뒤로 갈수록 덜 중요해진다.


HTML Lines
  •  <hr> tag는 page에 수평선을 만든다. 이것은 content를 구분 지을 때 사용된다.


HTML Comments
  • HTML comment는 document 안에서 code를 더 읽기 쉽고 이해하기 편하도록 작성될 수 있다. 브라우저는 이 comment를 무시하고 화면에 표시하지도 않는다.

HTML Tag Reference

  • <html>            - Defines an HTML document
  • <body>            - Defines the document’s body
  • <h1> to <h6>    - Defines HTML headings
  • <hr>               - Defines a horizontal line
  • <!-->               - Defines a comment

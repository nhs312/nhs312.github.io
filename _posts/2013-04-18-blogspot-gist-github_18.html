Gist GitHub

github는 사람들이 오픈소스를 공유하는 익히 유명한 사이트다.
여기서 public gist를 만들어서 내 포스트에 embed 시키는 방법이 있다.

github 사이트 : https://gist.github.com/
위 링크를 따라가면 로그인 하지 않아도 public gist를 만들 수 있다.

위와같이 로그인하지 않아도 code를 입력할 수 있다 programming language를 고를 수 있다.
코드를 붙여넣거나 작성한 뒤 우측 하단에 create public gist를 눌러 공개로 만들자.



create public gist를 누른 뒤에


<gist embed>
embed script를 복사해서 내 포스트에 붙여넣으면 완료.

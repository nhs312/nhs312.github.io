그래서 도움이 되고자 radtech 공홈에서 주문하는 과정을 처음부터 끝까지 포스팅 해본다.

Radtech Homepage

1. Radtech 공홈에서 프로모션 코드 얻기
홈페이지에서 프로모션 코드를 얻을 수 있다. 허허
공홈에 접속한 뒤, 상단 메뉴를 보면 두가지 메뉴가 보이는데..

<홈페이지 상단>

하나(Worldwide Shipping)는 해외 배송에 관한 안내 페이지 링크이고, 다른 하나(Join Now Save Today)는 프로모션 코드를 얻는 링크다.
링크를 누르면 이메일 주소와 이름을 입력하는 창이 나온다. 정보를 입력하고 메일을 기다리면 끝.
(처음으로 구매하는 사람에게만 해당하는 것인지 확인은 못했지만, 이 포스팅이 도움이 되는 사람은 처음 구매하는 것일테니 상관없지 않을까?)
정상적으로 입력했다면 다음과 같은 메일이 도착한다.


환영한다 어쩌구 저쩌구 하다가 중간에 보면
US free shipping : 미국내 무료 배송
50% off international shipping : 미국 외 국제 배송 50% 할인
20% off entire order : 상품 20% 할인
각각에 대한 프로모션 코드가 보인다.

2. 회원 가입과 주소 입력
실제 상품을 고르고 주문을 하기 위해서 회원 가입을 한다.
회원 가입 단계에서 주소를 입력하지 않아도 주문할 때 입력할 수 있으나 한번 입력해놓으면 편하니 나는 바로 입력했다.
영문 주소가 처음이라면 이번에 입력하는 주소를 복사해놓고 계속 사용하면 좋다.
(나는 자꾸 우편 번호를 까먹기 때문에..)
영문 주소 변환을 위한 우정국 주소 -> http://www.koreapost.go.kr/kpost/sub/subpage.jsp?contId=010101060000


위 입력 사항들을 채워넣자.
First Name : 이름
Last Name : 성
Organization/Company : 회사나 단체 이름
Email : 이메일 주소 - 주문 확인에 사용할 메일이니 주로 사용하는 메일이나 결제 전용 메일로..
Phone : 전화번호. 우리나라 국가코드를 넣어서 전화번호를 입력한다.
주소 부분. 
Address line 1 : 여기에는 가장 하위 주소, 상세한 주소를 입력하는 곳이다. e.g. 백석동 1439 번지
Address line 2 : address2에도 마찬가지로 상세 주소를 입력하면 되는데 굳이 line1에 들어가면 할 필요 없음.
City : 우리나라 시 단위에 해당. e.g. 고양시
State/Province : 우리나라 도 단위에 해당.
Postal Code : 우편 번호.
이렇게 주소를 채워 넣으면 끝. 실제로 영문 변환을 할 때 정확하지 않으면 어쩌나 하는 걱정이 있다. 우리나라에 물건이 들어오면 배달 지역을 구분할 때 알아볼 수 있으면 조금 스펠 틀려도 괜찮다고 한다.

3. 상품 주문과 결제
상품 주문은 별다른 도움이 필요 없다. 사려고 하는 물품을 보고 Add cart.
그리고 카트로 가서 결제 단계에 들어가자.
<결제 1단계>

결제를 누르면 위와 같이 가입시 미리 입력한 주소와 함께 배송 방법, 물건 확인 등이 뜬다.
USPS, FedEx가 있고 옵션은 비용과 기간을 함께 고려해서 선택하면 된다. 나는 가격은 얼마 차이 안나지만 하루이틀 값어치 할 것이란 기대로 FedEx Priority를 선택..
그리고 아래에는 주문한 물건들과 프로모션 코드를 입력하는 박스가 나온다.
저기서 내가 적용한 코드는 배송 50% off!! 물건을 두개 시켰기 때문에 20% off가 더 나을까 싶었지만 코드를 넣고 set을 눌러보면 그렇지 않더라..
아무튼 입력을 하고 창을 아래로 스크롤 해보면 다음과 같은 화면이..

<결제 1단계>

카드 정보와 배송 받을 주소를 입력하는 창이 나온다.
자신의 카드 중 해외 결제가 가능한 카드를 골라 정보를 입력한 뒤 가장 아래의 Continue..
그럼 2단계로 넘어간다.
<결제 2단계>

주문한 내용, 적용한 프로모션 코드, 받을 주소, 결제 수단(카드) 정보 등등이 최종적으로 보여지고 이상이 없다면 확인, 이상하다면 Make Changes.
<결제 3단계>

주문이 성공적으로 끝났음을 알려주는 화면.

4. 주문확인 E-mail
위 3의 과정을 잘 마치고 주문이 끝났다면 주문 확인 메일이 오게 된다.
<주문 확인 메일>

여기까지 왔다면 모든 과정이 끝난 것이고 이제 물건을 기다리면 된다.

이 포스팅의 결론은 영문 주소를 저장하자, 우편 번호도 저장하자. 해외에서 물건사자 정도가 되겠다.

아 맥북으로 처음 포스팅 해보는데 단축키나 스크린샷 편집이 익숙하지 않아서 좀 걸린 것 같다. 그런데도 기분이 좋다. 돌아가는 화면들이 이뻐서 그냥 기분이 좋다.

  • DOCTYPE은 document type을 선언하는 역할을 한다.
  • <html> </html> 사이는 웹 페이지를 설명하는 곳이다.
  • <body> </body> 사이는 페이지 컨텐츠가 실제로 보이는 곳이다.
  • <h1> </h1> 사이는 heading 부분으로 보여지는 곳이다.
  • <p> </p> 사이는 paragraph로 보여지는 곳이다.


HTML Tags

  • HTML markup tags는 보통 HTML tags라고 불린다.
  • HTML tags는 <html>과 같이 꺽쇠로 둘러싸인 키워드다.
  • HTML tags는 보통 <b> </b>와 같이 쌍을 이룬다.
  • 그 중 첫째 tag를 start tag, 둘째 tag를 end tag라고 한다.
  • End tag는 start tag에 /를 붙여서 쓰인다.
  • Start tags와 end tags는 opening tags와 closing tags로 불리기도 한다.


<tagname>content</tagname>


HTML Elements

  • HTML tags와 HTML elements는 같은 것을 설명하기도 한다.
  • 그러나 엄밀히는 HTML element는 start tag와 end tag를 포함해서 그 사이의 모든 것을  말한다.


<p>This is a paragraph.</p>


Web Browsers

  • 웹 브라우저의 목적은 HTML 문서를 읽고 웹 페이지로 보여주는 것이다. 브라우저는 HTML tags를 보여주지 않고 tags를 사용해서 페이지의 컨텐츠를 해석한다.


HTML versions과 DOCTYPE 선언

  • HTML - 1991
  • HTML+ - 1993
  • HTML 2.0 - 1995
  • HTML 3.2 - 1997
  • HTML 4.01 - 1999
  • XHTML 1.0 - 2000
  • HTML5 - 2012
  • XHTML5 - 2013



  • HTML5 - <!DOCTYPE html>
  • HTML 4.01 - <!DOCTYPE HTML PUBLIC “-//W3C//DTD HTML 4.01 Transitional//EN” http://www.w3.org/TR/html4/loose.dtd>
  • XHTML 1.0 - <!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN”  http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd>

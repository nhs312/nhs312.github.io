Writing HTML Using Notepad or TextEdit
  • HTML을 편집하기 위해 사용되는 툴은 다음과 같은 것들이 있다.

Adobe Dreamweaver / Microsoft Expression Web / CoffeeCup HTML Editor

  • 그러나 HTML을 배우기 위해서는 notepad나 textedit만으로도 충분하다. 배우는 단계에는 심플한 editor를 쓰는게 좋다.



다음 4단계를 따르면 notepad로 첫 웹페이지 샘플을 만들 수 있다.
  • Step 1: start Notepad

           윈도우 시작버튼 -> 모든 프로그램 -> 보조 프로그램 -> 메모장
           윈도우키 + R -> notepad 입력

  • Step 2: Edit Your HTML with Notepad

           HTML code를 notepad 안에 입력한다.

  • Step 3: Save HTML file

           Save 버튼을 눌러서 저장을 할 때 확장자를 .htm을 쓸 수도 있고 .html을 쓸 수도 있다.
이 두가지는 차이가 없으며 그냥 기호에 맞게 쓰면된다.

  • Step 4: Run the HTML in Your Browser

           저장한 파일을 웹브라우저에 드래그를 한다.
           아니면 브라우저에서 파일 열기 메뉴로 html 파일을 선택한다.
           그러면 브라우저에는 아까 만들었던 파일의 내용이 나타날 것이다.

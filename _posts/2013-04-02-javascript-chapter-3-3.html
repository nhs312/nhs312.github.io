Data Object 생성하기

Date object는 literal이 존재하지 않으므로 반드시 생성자를 통해서 object를 생성해야 한다.
1은 default 상태의 date object를 생성한다. 이 object의 시각은 생성된 시점의 시간으로 설정한다.
2는 날짜 문자열 값을 설정하여 object를 생성한다.
3은 년월일/시분초/밀리초의 형식으로 지정하고 싶을 때 사용한다.
4은 1970/01/01 00:00:00 부터의 경과 밀리초로 지정하는 방법이다.

Date object에는 다른 build-in object들과 같이 여러 method들이 있다.
주요 사용되는 것들은 getXXXX, setXXXX 와 같이 시간을 설정하고 얻는 method이다. 그외 다른 요소들은 검색을 통해 필요할 때 마다 사용한다.

1. Overview


역할

Google Analytics 를 사용하면 사용자가 모바일 기기, 플랫폼, 등을 통해서 나의 컨텐츠와 어떻게 관련되고 있는지 알 수 있다. 얼마나 자주 접속을 하며 어떤 컨텐츠를 가장 많이 보는지 또한 어떤 활동을 하는지 구분지어 볼 수 있다. 그리고 이 분석을 토대로 광고나 세일즈를 좀더 효과적으로 할 수 있게 도와주는 것이 Google Analytics 의 핵심 역할이다


2. 플랫폼




전체적인 흐름은 위와 같다.


Collection : user-interaction 데이터들을 수집한다.
Configuration : 이렇게 수집된 데이터들을 어떻게 처리할지 정해준다.
Processing : configuration 에서 설정한대로 데이터를 처리한다.
Reporting : 처리된 데이터들을 제공한다.

데이터를 수집하는 Collection 레벨에서는 Web Tracking, Android, iOS, Protocol 네 종류의 API 를 지원한다. (protocol 은 아직 beta 수준)
※ 현재 Picinit Web service 에서 필요한 Web Tracking 부분만 설명한다. (나머지 부분은 추후 업데이트 될 수 있음)

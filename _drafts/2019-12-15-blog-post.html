그래서 차이가 뭔가 알아보고 작성할 때 남들도 보기 좋게 만들어야겠다 싶어서 '세계적인 투자은행의 엑셀 업무 활용법'  라는 책을 읽었다.
얇아서 후딱 읽기 좋았고, 재무쪽 직군이 아니라 엑셀을 주력으로 쓰진 않을 터라 필요한 것들만 기록해두자.

1장 - 보기 쉬운 엑셀

1. 행높이는 18
2. 폰트 크기는 12
3. 영문자는 arial, 한글은 맑은 고딕 추천
4. 숫자는 콤마로 나눈다
5. 항목의 내역은 1열 들여서 쓴다
1. 들여쓰는 열 너비는 5(mac 기준)
2. 들여쓴 셀에서 ctrl + 방향키로 빠르게 이동
6. 단위는 같은 열에 맞춘다
7. 표 끝에 공백 열 (너비3)을 추가 해두면 보기 좋다
8. 테두리의 상하는 굵게, 그 밖은 가늘게
9. 글자는 왼쪽 정렬, 숫자는 오른쪽 정렬
1. 숫자가 들어있는 열의 항목명은 예외로 오른쪽 정렬
10. 필요없는 선은 지운다
11. A1부터 표를 시작하지 않는다. B2부터 해야 테두리 선도 보인다
12. 숫자에 색을 입힌다
1. 파랑 엑셀에 직접 입력한 숫자 - 변경 가능한 것은 최초 입력한 숫자뿐
2. 검정 계산식에 의해서 표시되는 숫자
3. 초록 다른 시트를 참조하고 있는 숫자
13. 배경색을 사용한다
1. 회사의 로고 컬러
2. 짙은색은 피하고 옅은 색으로
3. 자주 쓰이는 색은 옅은 파랑, 강조는 옅은 핑크
14. 눈금선은 보이지 않게 한다
1. 배경을 흰색으로 한다
2. 보기 → 눈금선 체크 해제
15. 그룹화 - 행/열을 숨긴다
1. 자신만 봐도 되는 셀 등에 대해서 사용
2. 숨기기 기능도 비슷한 역할을 하지만 플러스/마이너스 버튼이 없어서 셀이 숨겨져 있는지 알기 힘들다
3. 데이터 - 그룹
16. 숫자가 들어가지 않는 셀에는 N/A = 데이터가 들어가지 않는다. Not Applicable
17. 사용하지 않는 시트는 삭제한다
18. 포맷 정리
1. 데이터를 다 입력하고 포맷을 정리하는 방법은 비효율
2. 입력하면서 포맷을 정해가는게 좋음
3. 포맷은 문서화
4. 서로 포맷 오류를 지적
5. 정기적으로 확인 후 업데이트
19. 엑셀 표를 파워포인트 슬라이드 등에 붙일 때는 그림 상태로 붙여넣는다.
1. 표 복사
2. 홈 → 붙여넣기 → 선택하여 붙여넣기 → 그림 선택


1장의 내용들을 가지고 만든 엑셀 샘플

2장 - 실수 없는 엑셀

1. 실수를 하지 않기 위한 세가지 요령
1. 심플하게 알기 쉬운 계산을 철저히 한다
1. 수식에 숫자를 직접 입력하지 않는다.
2. 너무 긴 수식은 NG - 본인이 머리가 좋아도 남이 알기 어려우면 건드릴 수 없다
3. 시트의 구성을 알기 쉽게 차트로 정리
1. 시트는 왼쪽에서 오른쪽으로 계산 - 계산의 흐름 순으로 결론이 마지막에 가도록
2. 시트를 그룹화 해서 색으로 구분
3. 시트는 반드시 표시
2. 계산 체크를 철저히 한다
1. 계산 체크는 매우 중요
1. 계산할 때마다 체크 / 모든 셀을 체크 / 체크에 들이는 시간 = 계산에 걸리는 시간
2. F2 로 수식 체크
3. 셀 추적 - 참조되는 셀을 보여주기 때문에 비교하기 쉽다. 수식 - 참조셀 추적(단축키 사용)
2. 수치의 추이는 선으로 체크 (단축키 사용)
3. 팀워크를 철저히 한다
1. 계산은 한 명, 파일 한 개에 한다
2. 두 명이 체크할 때 중요한 포인트는 직접 수정하는 게 아니라 담당자에게 피드백을 줘서 수정하게 한다
3. 엑셀 파일은 다른 이름으로 저장하고 덧씌우지 않는다
4. 지난 파일은 현재 위치에서 old 폴더를 만들어서 저장
5.
2. 시트의 수는 가능한 적게 한다
3. 숫자의 출처는 명확하게 표기한다
4. 심플한 계산을 소중하게 여기는 팀 문화가 필요
5. 실수를 줄이기 위해서는 팀장의 노력이 필요
1. 계산에 대해 질문
2. 충분한 엑셀 작업 시간 확보
3. 계산 실수에 대한 최종적인 책임은 리더가 진다

3장 - 빠른 엑셀

1. 포맷과 계산 체크를 철저히 한다
2. 단축키 활용
1. 셀 서식
2. 시트 전체 선택
3. 글자 → 글자로 이동
4. 데이터 끝까지 선택
5. 다른 시트 이동
6. 행/열 삽입
7. 참조셀로 이동
8. 포맷 (배경색/문자색/폰트)
9. 정렬
10. 그룹화
11. 셀 추적
12. 꺾은 선 그래프
13. 선택하여 붙여넣기
3. 같은 작업은 F4로 반복한다
4. 숫자로 생각할 기회를 늘린다

Mac vs Windows 관점에서 엑셀은 윈도우에서 작업하는게 좋다. 맥에서도 어느정도 가능하지만 고급 기능들에 대한 단축키가 다르고 없는 경우도 많기 때문이다.
